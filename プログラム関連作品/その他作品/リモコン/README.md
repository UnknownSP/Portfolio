# リモートコントローラ

## コード置き場及び詳細

- [RemoteControl](https://github.com/UnknownSP/RemoteControl)

## 概要

Discordを介して外部から部屋の電気やエアコンなどの操作をすることができます。 操作はArduinoに搭載した赤外線LEDによって行われます。

TV・部屋の電気・エアコン・ダイソン扇風機・Webカメラ・環境センサ等の操作をDiscordを介して外出先からでも操作ができます。
